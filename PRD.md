# PRD – 한화생명 사내 AI 어시스턴트

## 📌 프로젝트 개요

### 제품명
한화생명 AI 어시스턴트 (가칭: "김한화")

### 비전
"만능 AI 동료" - 모든 업무를 함께하는 똑똑한 파트너

### 핵심 가치
- **통합성**: 흩어진 사내 정보를 하나로 연결
- **효율성**: 반복 업무 자동화로 생산성 향상
- **친밀감**: 동료처럼 편안한 대화형 인터페이스

### 타겟 사용자
한화생명 전 임직원 (신입사원부터 임원까지)

## 🎯 문제 정의

### 현재 임직원들이 겪는 실제 문제들
- "대리님! 저번에 정비실 담당자분 성함 기억나세요?"
- "형광등이 나갔는데 이건 어디에 말을해야할까요?"
- "이번에 자기개발 지원제도 1년에 얼마까지였죠?"
- "저번에 팀장님께 보고드린 보고서 양식 공유받을 수 있을까요?"
- "사내교육 필수로 들어야하는거 링크 아세요?"

## 🚀 핵심 기능

### 1. 통합 정보 검색 시스템

#### Pain Points
- 사내 시스템이 분산되어 있어 정보 찾기 어려움
- 시스템마다 다른 비밀번호로 접근성 저하
- 담당자 정보 파악의 어려움

#### Solutions
- **통합 검색 엔진**: 모든 사내 데이터를 하나의 인터페이스로 검색
- **자연어 처리**: "오늘 구내식당 메뉴 뭐야?" 같은 일상 대화로 정보 획득
- **개인화된 정보 제공**: 연차, 급여, 복지 포인트 등 개인 정보 즉시 조회
- **보안 기능**: AI 전용 비밀번호 설정으로 안전한 개인정보 접근

#### 예상 효과
- 정보 검색 시간 80% 단축
- 시스템 접근성 90% 향상

### 2. 지능형 보고서 생성

#### Pain Points
- 부서별/상황별 다른 보고서 양식으로 인한 피로도
- 반복적인 데이터 입력 작업
- 범용 AI 도구의 한계 (기업 아이덴티티 미반영)

#### Solutions
- **자동 콘텐츠 생성**: 제목만 입력하면 관련 데이터와 출처 자동 삽입
- **한화생명 템플릿**: 기업 CI/BI 자동 적용 (색상, 폰트, 레이아웃)
- **맥락 인식 조언**: 
  - "팀장님께는 이 지표가 중요하실 거예요"
  - "작년 대비 성장률 추가하면 좋을 것 같아요"
- **사내 데이터 연동**: 실시간 KPI, 프로젝트 현황 자동 반영

#### 경쟁 우위
| 구분 | 일반 AI 도구 | 한화 AI 어시스턴트 |
|------|-------------|-------------------|
| 역할 | 단순 도구 | 업무 파트너 |
| 데이터 | 일반 정보 | 사내 데이터 통합 |
| 스타일 | 범용 템플릿 | 한화생명 맞춤형 |
| 상호작용 | 일회성 | 지속적 대화 |

#### 예상 효과
- 보고서 작성 시간 50% 단축
- 데이터 정확도 95% 이상

### 3. 스마트 일정 관리

#### Pain Points
- 다부서 협업 시 일정 조율의 복잡성
- 반복적인 메일/메신저 커뮤니케이션
- 회의실 예약의 번거로움

#### Solutions
- **자동 일정 매칭**: 참석자 캘린더 분석으로 최적 시간대 제안
- **회의실 자동 예약**: 인원, 장비 필요사항 고려한 자동 배정
- **참석자 자동 초대**: 이메일/캘린더 초대장 자동 발송
- **부가 서비스**: 음료 주문, 자료 사전 공유 등

#### 예상 효과
- 일정 조율 시간 70% 단축
- 회의 준비 효율성 85% 향상

## 🔧 추가 기능 모듈

### 업무 효율화 기능

#### 1. 업무 히스토리 추적
- **기능**: 화면 스크린샷으로 담당자 찾기
- **활용**: 프로젝트 인수인계, 담당자 연결

#### 2. 자동 업무일지 작성
- **기능**: 캘린더 기반 업무 요약
- **활용**: 일일/주간/월간 리포트 자동 생성

#### 3. 사내 규정 실시간 체크
- **기능**: 복지, 휴가, 경비 규정 즉답
- **활용**: HR 문의 감소, 셀프 서비스 확대

### 협업 강화 기능

#### 1. 사내 네트워킹 매칭
- **기능**: 전문가 연결, 동아리 추천
- **활용**: 지식 공유 활성화

#### 2. 팀 회식 플래너
- **기능**: 장소 추천, 법인카드 한도 체크
- **활용**: 팀 빌딩 활동 지원

#### 3. 익명 소원함
- **기능**: 익명 건의사항 전달
- **활용**: 조직 문화 개선

### Fun & Engagement

#### 1. 오늘의 운세
- 팀별, 파트별 운세 순위

#### 2. 칭찬 도장 릴레이
- 이달의 칭찬왕 선정

#### 3. 빠른 사다리 타기
- 팀내 회식, 랜덤 인원 뽑기

## 💡 향후 확장 아이디어

### 개인화 서비스
- **나의 AI 프로필 자동 갱신**: 강점/약점 분석, 성향 파악
- **MBTI 기반 커뮤니케이션 가이드**: 상대방 성향 맞춤 소통
- **경력 개발 로드맵**: 개인 맞춤형 성장 경로 제안

### 스마트 오피스
- **실내 환경 제어**: 온도, 조명 조절 요청
- **혼잡도 안내**: 엘리베이터, 식당, 회의실 실시간 정보
- **스마트 좌석 예약**: 집중 공간, 조용한 자리 추천

### 사내 마켓플레이스
- **중고 마켓**: 사진 기반 자동 상품 등록
- **소모임 플랫폼**: 관심사 기반 모임 매칭

### 프로젝트 인텔리전스
- **기술 스택 추적**: 프로젝트별 사용 기술 데이터베이스
- **유사 프로젝트 분석**: 성공/실패 사례 학습
- **자동 문서화**: 프로젝트 히스토리 자동 정리

## 🎯 실제 사용 시나리오

### 시나리오 1: 보고서 작성
```
사용자: "이번 달 자동차보험 클레임 현황 보고서 만들어줘"
김한화: "네! 3월 자동차보험 클레임은 전월 대비 15% 증가했어요. 
        주요 원인과 대응 방안을 포함한 보고서 초안을 작성해드릴게요."
```

### 시나리오 2: 일정 조율
```
사용자: "내일 오후에 기획팀과 미팅 잡아줘"
김한화: "기획팀 3명과 모두 가능한 시간은 오후 2-3시예요. 
        회의실 B를 예약하고 참석자들께 초대장을 보낼까요?"
```

### 시나리오 3: 정보 검색
```
사용자: "연차 얼마나 남았어?"
김한화: "현재 5일 남았습니다. 내년 1월에 15일이 추가될 예정이에요."
```
